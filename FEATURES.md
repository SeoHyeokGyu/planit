```
입력: "매일 책 읽기"
출력: {
  title: "매일 책 읽기 30분 챌린지",
  description: "바쁜 일상 속에서도...",
  tags: ["#독서", "#자기계발", "#습관"],
  category: "STUDY"
}
```

### 2. AI 인증샷 분석기
```
사진 업로드 시 자동 분석
운동 사진 → "운동 인증이 확인되었습니다!"
물 마시기 사진 → "물 마시기 인증이 확인되었습니다!"
```

### 3. AI 동기부여 코치
```
사용자 인증 후 자동 메시지 생성
"5km 완주했군요! 꾸준함이 멋집니다!"
```
