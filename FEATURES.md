# AI 기반 소셜 챌린지 트래커 - 웹 기능 목록

> 작은 성취를 실시간으로 공유하고, AI가 다음 도전을 추천해주는 동기부여 플랫폼 (웹 기반)

---

## 기능 목록

### 1. 사용자 관리 (User Management)
- [ ] 회원가입 페이지 (이메일, 비밀번호, 닉네임)
- [ ] 이메일 인증 (이메일 발송 + 인증 링크)
- [ ] JWT 기반 로그인/로그아웃
- [ ] 프로필 페이지 (프로필 사진, 소개, 레벨, 배지 표시)
- [ ] 프로필 수정 페이지
- [ ] 비밀번호 변경 페이지
- [ ] 회원 탈퇴 기능

### 2. 챌린지 관리 (Challenge Management)
- [ ] 챌린지 목록 페이지 (카드 형식, 필터링, 정렬)
- [ ] 챌린지 상세 페이지 (설명, 참여자, 인증 목록, 통계)
- [ ] 챌린지 생성 페이지 (제목, 설명, 카테고리, 기간, 난이도, 태그)
- [ ] 챌린지 수정 페이지 (작성자만)
- [ ] 챌린지 삭제 기능
- [ ] 챌린지 참여 버튼 (모달 확인)
- [ ] 참여 챌린지 탈퇴 기능
- [ ] 챌린지 키워드 검색 (실시간 검색)
- [ ] 조회수, 참여자 수 실시간 표시

### 3. 실시간 피드 ⭐ 핵심 기능 (Real-time Feed)
- [ ] 피드 페이지 (참여 중인 챌린지의 인증들)
- [ ] SSE 연결 (EventSource API 사용)
- [ ] 실시간 토스트 알림 ("OO님이 방금 인증했습니다!")
- [ ] 알림 3초 자동 사라짐
- [ ] 연결 상태 표시 (연결됨/끊김)
- [ ] 자동 재연결 로직 (3초 간격)
- [ ] 로드 더 보기 (무한 스크롤 또는 더보기 버튼)
- [ ] 인증 정렬 옵션 (최신순, 좋아요 많은순)

### 4. 사진 업로드 (Photo Upload)
- [ ] 인증 작성 페이지 (텍스트 + 사진)
- [ ] 드래그앤드롭 업로드 영역
- [ ] 파일 선택 버튼
- [ ] 사진 미리보기 (업로드 전)
- [ ] 업로드 진행률 표시 (프로그레스 바)
- [ ] 썸네일 자동 생성 (목록에서 표시)
- [ ] 이미지 압축 (클라이언트 사이드)
- [ ] 파일 검증 (타입: jpg, png, webp / 크기: 5MB 이하)
- [ ] 업로드된 사진 삭제 버튼
- [ ] 사진 갤러리 뷰 (그리드/리스트)

### 5. AI 추천 엔진 ⭐ 핵심 기능 (AI Recommendation Engine)
- [ ] 추천 페이지 (맞춤 추천 챌린지 목록)
- [ ] 5가지 추천 방식:
  - 단순 쿼리 기반: "가장 핫한 챌린지" (참가자 수 TOP)
  - 콘텐츠 기반: "관심 태그 기반" (유사 태그)
  - 협업 필터링: "나와 비슷한 사용자" (공통 참여 챌린지)
  - 개인화: "당신의 활동 패턴에 맞춰" (난이도 선호)
  - 트렌드: "최근 뜨는 챌린지" (24시간 급상승)
- [ ] 각 추천마다 "왜 추천했나요?" 설명 표시
- [ ] 추천 새로고침 버튼
- [ ] 추천 챌린지 카드 (이미지, 제목, 참여자 수, 평점)
- [ ] 탭 네비게이션 ("맞춤 추천" / "인기 챌린지" / "트렌딩")
- [ ] 로딩 상태 표시 (스켈레톤 로더)

### 6. AI 챌린지 생성기 (AI Challenge Generator)
- [ ] AI 생성 페이지
- [ ] 키워드 입력 필드 (예: "매일 책 읽기")
- [ ] 생성 버튼 (로딩 스피너)
- [ ] 생성된 챌린지 미리보기:
  - 자동 생성된 제목
  - 자동 생성된 설명
  - 자동 생성된 태그
  - 자동 추천 카테고리
- [ ] 수정 기능 (제목, 설명, 태그 수정 가능)
- [ ] 생성 완료 버튼 (챌린지 즉시 생성)
- [ ] 취소 버튼

### 7. AI 인증 분석 (AI Certification Analysis)
- [ ] 인증 업로드 시 백그라운드 분석
- [ ] 분석 진행 중 표시 (로딩)
- [ ] 분석 완료 후 배지 자동 표시:
  - "AI 인증" 배지 표시
  - 분석 결과 메시지 (예: "운동 인증이 확인되었습니다!")
- [ ] 분석 실패 시 경고 메시지
- [ ] 인증 상세 페이지에서 AI 분석 결과 표시
- [ ] AI 신뢰도 점수 표시 (%)

### 8. AI 동기부여 코치 (AI Motivational Coach)
- [ ] 인증 완료 후 AI 격려 메시지 팝업
- [ ] 메시지 내용 (개인화된 격려)
  - 사용자의 활동 패턴 반영
  - 현재 스트릭 / 레벨 기반
  - 다양한 표현으로 매번 다른 메시지
- [ ] 메시지 공유 버튼 (SNS 공유)
- [ ] 메시지 저장 기능
- [ ] 알림으로도 전송

### 9. 팔로우/팔로워 (Follow System)
- [ ] 사용자 프로필 페이지에서 팔로우 버튼
- [ ] 팔로우/언팔로우 토글
- [ ] 팔로워 수 표시
- [ ] 팔로잉 수 표시
- [ ] 팔로워 목록 모달 (이름, 프로필 사진)
- [ ] 팔로잉 목록 모달
- [ ] 팔로잉 피드 페이지 (팔로우한 사용자들의 인증 모음)
- [ ] 팔로잉 피드의 실시간 업데이트

### 10. 댓글 & 좋아요 (Comments & Likes)
- [ ] 인증글 상세 페이지에서 댓글 영역
- [ ] 댓글 입력 필드 (텍스트)
- [ ] 댓글 전송 버튼
- [ ] 댓글 목록 (프로필 사진, 닉네임, 내용, 시간)
- [ ] 댓글 페이징 (또는 더보기)
- [ ] 자신의 댓글 삭제 버튼
- [ ] 좋아요 버튼 (하트 아이콘)
- [ ] 좋아요 카운트 (실시간 업데이트)
- [ ] 좋아요 목록 모달 (좋아요 누른 사용자들)
- [ ] 좋아요 애니메이션 (버튼 클릭 시)

### 11. 알림 (Notifications)
- [ ] 상단 네비게이션에 알림 벨 아이콘
- [ ] 미읽음 알림 카운트 배지
- [ ] 알림 드롭다운 패널 (최근 5개)
- [ ] 알림 목록 페이지 (전체 알림)
- [ ] 알림 타입별 아이콘:
  - 팔로우: 👤
  - 댓글: 💬
  - 좋아요: ❤️
  - 배지 획득: 🏅
  - 레벨 업: ⬆️
- [ ] 알림 읽음 처리 (클릭 시)
- [ ] 알림 삭제 버튼
- [ ] 모든 알림 읽음 버튼
- [ ] 알림 설정 페이지 (수신 여부 토글)
- [ ] 푸시 알림 권한 요청 (브라우저)

### 12. 소셜 로그인 (Social Login)
- [ ] 로그인 페이지의 소셜 로그인 버튼:
  - Google 로그인 버튼
  - Kakao 로그인 버튼
- [ ] OAuth 팝업/리다이렉트 처리
- [ ] 소셜 계정 정보 자동 입력 (이름, 이메일, 프로필 사진)
- [ ] 기존 계정과 소셜 계정 연동 페이지
- [ ] 프로필 페이지에서 연동된 소셜 계정 표시
- [ ] 소셜 계정 연동 해제 버튼
- [ ] 소셜 로그인 실패 메시지

### 13. 포인트/레벨 시스템 (Points & Level System)
- [ ] 프로필에 현재 레벨 표시 (아바타 옆)
- [ ] 레벨 진행률 바 (0% ~ 100%)
- [ ] 현재 포인트 표시 (다음 레벨까지 필요 포인트)
- [ ] 레벨 업 애니메이션 + 축하 모달
- [ ] 포인트 획득 로그 페이지:
  - 인증 작성: +10 포인트
  - 댓글: +2 포인트
  - 좋아요: +1 포인트
  - 배지 획득: +5 포인트
- [ ] 포인트 히스토리 (최근 활동)
- [ ] 랭킹 페이지 (포인트 기반 순위)

### 14. 배지 시스템 (Badge System)
- [ ] 프로필에 획득한 배지 표시 (아이콘 그리드)
- [ ] 배지 모음 페이지 (전체 배지 목록)
- [ ] 배지 종류:
  - "첫 인증" (첫 인증 시)
  - "7일 연속" (7일 연속 인증)
  - "100 포인트" (100 포인트 달성)
  - "10팔로워" (팔로워 10명)
  - "50인증" (인증 50개)
  - 등급별 (브론즈, 실버, 골드, 플래티넘)
- [ ] 배지 설명 (호버 시 팝오버)
- [ ] 배지 획득 시간 표시
- [ ] 미획득 배지 회색 표시 (획득 조건 표시)
- [ ] 배지 획득 알림

### 15. 스트릭 (Streak System)
- [ ] 프로필에 현재 스트릭 표시 (큰 숫자 + 불 아이콘)
- [ ] 최고 스트릭 기록 표시
- [ ] 스트릭 캘린더 (최근 30일)
  - 인증한 날: 초록색
  - 인증 안 한 날: 회색
- [ ] 스트릭 끊김 경고 (오늘 미인증 시)
- [ ] 스트릭 통계 페이지

### 16. 랭킹 (Ranking/Leaderboard)
- [ ] 랭킹 페이지 (상위 100명)
- [ ] 필터 탭 ("이번 주" / "이번 달" / "전체")
- [ ] 순위표 (1-100위):
  - 순위
  - 프로필 사진
  - 닉네임
  - 포인트
  - 배지 개수
- [ ] 본인 순위 하이라이트 (배경색 다르게)
- [ ] 본인의 상세 순위 표시 (n위 중 몇 등)
- [ ] 순위 변동 표시 (↑↓)
- [ ] 실시간 순위 업데이트 (WebSocket)

### 17. 통계/분석 (Statistics & Analytics)
- [ ] 개인 통계 페이지 (대시보드):
  - 총 인증 수
  - 총 포인트
  - 현재 레벨
  - 획득 배지 수
  - 현재 스트릭
  - 평균 인증 시간
- [ ] 챌린지 통계 (챌린지 상세 페이지):
  - 참여자 수
  - 인증 수
  - 완료율 (%)
  - 인기도 (뷰 수)
- [ ] 차트 시각화 (Chart.js / Recharts):
  - 일별 인증 추이 (선 그래프)
  - 주별 포인트 (막대 그래프)
  - 카테고리별 참여도 (도넛 차트)
- [ ] 기간 필터 (1주, 1개월, 3개월, 전체)
- [ ] CSV 다운로드 기능

### 18. 고급 검색 (Advanced Search)
- [ ] 챌린지 검색 페이지
- [ ] 검색 필터:
  - 카테고리 체크박스 (운동, 공부, 취미, 건강, 창작 등)
  - 난이도 라디오 버튼 (EASY / MEDIUM / HARD)
  - 기간 레인지 슬라이더 (1일 ~ 90일)
  - 상태 필터 (진행 중 / 종료된 / 예정된)
- [ ] 정렬 옵션 (인기도 / 신규순 / 참여자 많은순 / 종료 예정)
- [ ] 실시간 검색 결과 업데이트
- [ ] 필터 태그 표시 (적용된 필터)
- [ ] 필터 초기화 버튼
- [ ] 검색 결과 수 표시

### 19. 개인 대시보드 (Personal Dashboard)
- [ ] 대시보드 홈 페이지:
  - 현재 레벨 카드 (프로필 사진, 레벨, 진행률)
  - 포인트 현황 카드 (현재 / 다음 레벨)
  - 배지 갤러리 (최근 3개)
  - 스트릭 카드 (현재 스트릭, 최고 기록)
- [ ] 참여 중인 챌린지 (카드 그리드)
  - 챌린지 사진
  - 진행도 바
  - 본인의 인증 수 / 남은 기간
- [ ] 최근 활동 피드 (최근 인증 5개)
  - 시간 표시
  - 스크린샷/사진 썸네일
- [ ] 추천 챌린지 섹션 (AI 추천)
- [ ] 빠른 액션 버튼 (+ 챌린지 참여, + 인증 작성)
- [ ] 모바일 반응형 (세로/가로)

## AI 기능 (차별화 포인트)

### 1. AI 챌린지 생성기
```
입력: "매일 책 읽기"
출력: {
  title: "매일 책 읽기 30분 챌린지",
  description: "바쁜 일상 속에서도...",
  tags: ["#독서", "#자기계발", "#습관"],
  category: "STUDY"
}
```

### 2. AI 인증샷 분석기
```
사진 업로드 시 자동 분석
운동 사진 → "운동 인증이 확인되었습니다!"
물 마시기 사진 → "물 마시기 인증이 확인되었습니다!"
```

### 3. AI 동기부여 코치
```
사용자 인증 후 자동 메시지 생성
"5km 완주했군요! 꾸준함이 멋집니다!"
```

## 데이터 저장소 전략

| 데이터 | 저장소 | 이유 |
|--------|--------|------|
| User, Challenge, Certification | **PostgreSQL** | 정규화된 관계형 데이터 |
